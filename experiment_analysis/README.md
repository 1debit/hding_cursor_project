# 🧪 Experiment Analysis

## 🎯 项目概述

**实验分析工具集** - 为Chime产品团队提供专业的A/B测试分析、统计建模和实验结果评估的完整解决方案。

## 🚀 核心功能

### 1. A/B测试功效分析
- **样本量计算**: 基于效应大小和统计功效
- **功效分析**: 评估实验检测能力
- **置信区间**: 提供结果可靠性评估

### 2. 统计建模
- **假设检验**: t检验、卡方检验、ANOVA等
- **效应大小**: Cohen's d、相对风险等指标
- **多重比较**: 控制家族错误率

### 3. 实验结果分析
- **描述性统计**: 均值、标准差、分布分析
- **推断性统计**: 显著性检验、置信区间
- **可视化报告**: 图表和交互式仪表板

## 📊 分析模板

### 功效分析模板
- **输入参数**: 效应大小、α水平、β水平
- **输出结果**: 所需样本量、实际功效
- **可视化**: 功效曲线、样本量曲线

### 结果分析模板
- **数据预处理**: 异常值检测、缺失值处理
- **统计检验**: 选择合适的统计方法
- **结果解释**: 业务意义和统计意义

## 📈 分析指标

### 统计功效
- **高功效**: ≥0.8 (推荐)
- **中等功效**: 0.5-0.8
- **低功效**: <0.5 (不推荐)

### 效应大小
- **小效应**: d = 0.2
- **中等效应**: d = 0.5
- **大效应**: d = 0.8

### 显著性水平
- **标准**: α = 0.05
- **严格**: α = 0.01
- **宽松**: α = 0.10
